Descrição

UPL tem como objetivo ser simples, compacta e portável. Ou seja, ser pequena, uma sintaxe resumida e com possibilidade para gerar códigos para outras linguagens. Contendo apenas um núcleo composto por comandos de controle de fluxo, funções e a possibilidade de executar alguma função da API do sistema.

Gera código para as linguagens:

  • C - Desenvolvimento de aplicações em geral.
  • PHP - Desenvolvimento Web.

Outras características:

  • Case-sensitive
  • Estruturada

Funções e módulos

Para declaração de funções usamos o comando def e todo código entre as chaves { e } é parte da mesma. E para declaração de módulos usamos o comando mod no início do arquivo de código, onde o mesmo possui funcionamento igual ao comando include da linguagem C.

/* Arquivo de exemplo (exemplo.upl). */

mod pacote // Define nome do arquivo sem extensão (Ex: para 'pacote.upl' define-se 'pacote').
mod pacoteA, pacoteB // Definição de múltiplos pacotes separados por vírgula.

// Core do módulo.
// Definição como public, onde a função pode ser acessada por outros módulos.
public functionCore{
    obj = pacoteCore(), say "{obj}\n" // Separação de comandos com vírgula, execução de função do módulo 'pacote' e exibição dos dados.

    functionZero() // Chamada simples.
    variavel = functionZero() // Chamada com retorno para variável.
    variavel = functionTwo(10, 20) // Chamada com passagem de parâmetros.
    variavel1 = functionTwo(10, 20), variavel2 = functionThree(10, 20, "N") // Chamadas separadas por vírgula.

    ret true
}

// Definição como static, onde a função pode ser acessada somente pelo próprio módulo.
static functionZero{/* Code here... */}

// Outros exemplos...

{ /* Code here... */ }

     ret valor1 + valor2
}

Comandos de controle de fluxo

Comando Descrição
if Se condição for verdadeira executa bloco de código.

if (CONDIÇÃO) {
     // Bloco de código...
}

if (CONDIÇÃO)
     // Executa primeiro bloco de código encontrado por não fazer uso das chaves.
el Quando vazio executa bloco padrão (comportamento equivalente ao comando else da linguagem C), o bloco apenas é executado se não existir nenhuma ocorrência anterior.

el {
     // Bloco de código...
}

Quando passado o bloco condicional comporta-se como comando else if da linguagem C.

el (CONDIÇÃO) {
     // Comportamento equivalente ao comando else if da linguagem C).
}

Funcionamento como da estrutura de comandos if, else if e else da linguagem C.

if (CONDIÇÃO) {
     // Bloco de código...
} el (CONDIÇÃO) {
     // Bloco de código...
} el {
     // Bloco de código...
}
for Mesmo funcionamento que o comando na linguagem C, mas com alguns adicionais. O bloco condicional é apenas necessário declara-lo se realmente for utilizar o mesmo. Por exemplo, para um bloco com repetições infinitas podemos fazer da seguinte maneira.

for {
     // Bloco de código...
}

// Imprimir números de 0 a 9...
for (a=0; a<=9; a++)
     // Bloco de código...

// Utilização de múltiplas variáveis.
for (a=0,b=1,c=2; a<=9; a++,b+=2,c+=3)
     // Bloco de código...
break Encerra a execução do comando for, pulando para o bloco de código seguinte.
continue Continua execução do comando for, indo para ocorrência seguinte.


Chamadas a funções da API do sistema

Especifica-se o nome da função e seus parâmetros.

MessageBoxA(NULL, "Minha mensagem!", "Meu titulo!", MB_OK)

// Quando se trata de variáveis...
messageA = "Msg..", messageB = "Msg..."
MessageBoxA(NULL, messageA, messageB, MB_OK)

Para fazer uso de funções da API do sistema, deve-se definir o protótipo da função no arquivo de configuração 'prototype.cnf'.

int WINAPI MessageBox(
     _In_opt_ HWND hWnd,
     _In_opt_ LPCTSTR lpText,
     _In_opt_ LPCTSTR lpCaption,
     _In_       UINT uType
);

LRESULT CALLBACK MessageProc(
     _In_ int code,
             WPARAM wParam,
     _In_ LPARAM lParam
);


Caracteres especiais, constantes e operadores

Caracteres especiais
Caractere Descrição
= Atribuição de valores.
// Comentário de uma única linha ou parte da mesma.
/*, */ Comentário de múltiplas linhas, com início em /* e término em */.
new-line Nova linha, finalização de bloco de comando.
, (vírgula) Vírgula, finalização de definição de variável (separação para definição de múltiplas variáveis).
{, } Chaves, define bloco de código, com início em { e término em }.
" (aspas duplas) String, define uma sequência de caracteres, com início na primeira ocorrência encontrada e término na seguinte.

Constantes
Para fazer uso de constantes é necessário declará-las no arquivo de configuração 'constants.cnf'. Definição e valor separados por espaço.

MB_OK 0
TRUE 1
FALSE 0
true 1
false 0

Operadores matemáticos
Operador Descrição
+ Soma valores.
- Subtrai valores.
* Multiplica valores.
/ Divide valores.
% Resto de divisão.

Operadores para incremento e decremento
Operador Descrição
++ Incrementa valor em um.
-- Decrementa valor em um.
+= Soma e atribui valor.
-= Subtrai e atribui valor.
*= Multiplica e atribui valor.
/= Divide e atribui valor.

Operadores condicionais
Operador Descrição
== Verifica se valores da esquerda e direita são idênticos.
< Verifica se valor da esquerda é menor que valor da direita.
> Verifica se valor da esquerda é maior que valor da direita.
<= Verifica se valor da esquerda é menor ou idêntico ao valor da direita.
>= Verifica se valor da esquerda é maior ou idêntico ao valor da direita.
!= Verifica se valor da esquerda não é igual ao valor da direita.

Operadores lógicos
Operador Descrição
&& E lógico, equivalente ao comando na linguagem C.
|| OU lógico, equivalente ao comando na linguagem C.
! NÃO lógico, equivalente ao comando na linguagem C.

Descrição

Short Syntax é um conversor de código, um código responsável por ler algum
código escrito utilizando a sintaxe descrita aqui neste texto e gerar outro código
equivalente de saída em alguma linguagem suportada pelo mesmo.

É compacta: suas características foram simplificadas ao máximo,
deixando a mesma com uma sintaxe pequena e resumida. É estruturada. É portável: é
possível a geração de código para plataformas diferentes. Segue abaixo as linguagens
utilizadas e suas respectivas plataformas de destino:

  • PHP - Desenvolvimento Web.
  • Java - Desenvolvimento mobile.
  • Perl, C, ASM - Desenvolvimento desktop e programação de baixo nível

É flexível e poderosa: ao gerar código nativo para outras linguagens, traz consigo mais
poder e flexibilidade, permitindo o uso de características das linguagens de destino.
Podendo ser vista como um adicional para outras linguagens.

Outras características:

  • Case-sensitive
  • Estruturada


Variáveis

As variáveis devem sempre, ao ser declaradas, também inicializadas com algum valor.
Segue abaixo alguns exemplos.

// Definição de variável, aceita qualquer tipo de dado.
variavel = alloc(256)       // Internamente é feito a alocação de espaço na
                                    // memória e em seguida é atribuído o valor zero a variável.

// Outros exemplos...
variavel = null                    /* Zera variável, seta todos os dados para 0 (zero)
                                            e libera espaço na memória. */
variavel = "String here!"      /* Simples C string. */
variavel = 100                    /* Valor inteiro. */
variavel = alloc(256)           /* Armazena 256 bytes na memória. */
variavel = funcN()              /* Armazena retorno da função na variável. */

/* Múltiplas variáveis em uma única linha. */
variavelA = 10, variavelB = 0, variavelC = "String here!", variavelD = alloc(256)
variavelE = null, variavelF = funcN(), variavelG = 0

Ao inicializar uma variável com uma cadeia de caractere ou algum valor numérico,
é calculado o tamanho necessário, feita a alocação e copiado os dados para memória.

Ao inicializar ou atribuir a uma variável com o valor null, é alocado (ou re-alocado)
apenas o tamanho de seu tipo(ou o tamanho necessário) e lhe atribuido o valor 0.

Quando a variável recebe como atribuição a função alloc, a variável em questão
terá seu tamanho de armazenamento re-alocado de acordo com o valor passado
como parâmetro para função.

Quando a variável recebe qualquer outra função como atribuição de valor; seu
valor de retorno é armazenado na variável.

Para declarar múltiplas variáveis em uma única linha, basta separa-las por vírgula.

Controle de fluxo

Comando Descrição
if Se condição for verdadeira executa bloco de código.

if (CONDIÇÃO) {
     // Bloco de código...
}
elif Quando vazio executa bloco padrão (comportamento como do comando else da linguagem C), o bloco apenas é executado se não existir nenhuma ocorrência anterior.

elif {
     //code here!
}

Quando é passado parâmetros, ou seja, algum bloco de condições, a sequência de ifs finaliza com a primeira ocorrência encontrada.

elif (CONDIÇÃO) {
     // code here... (comportamento como
     // do comando else if da linguagem C).
}
for Mesmo funcionamento que o comando na linguagem C, mas com alguns adicionais. O bloco condicional é apenas necessário declara-lo se realmente for utilizar o mesmo. Por exemplo, para um bloco com repetições infinitas podemos fazer da seguinte maneira.

for {
     // code here!
}

// Imprimir números de 0 a 9...
for (a=0; a<=9; a++)
     // Code here...

// Utilização de múltiplas variáveis.
for (a=0,b=1,c=2; a<=9; a++,b+=2,c+=3)
     // Code here...
break Encerra a execução do comando for, pulando para o bloco de código seguinte.
continue Continua execução do comando for, indo para ocorrência seguinte.

Operadores, constantes e determinados caracteres.

Caracteres
Caractere Descrição
= Atribuição de valores.
// Comentário de uma única linha ou parte da mesma.
/*, */ Comentário de múltiplas linhas, com início em /* e término em */.
new-line Nova linha, finalização de bloco de comando.
, (vírgula) Vírgula, finalização de definição de variável (separação de definição de múltiplas variáveis).
{, } Chaves, define bloco de código, com início em { e término em }.
" (aspas dupla) String, define uma sequência de caracteres, com início na primeira ocorrência encontrada e término na seguinte.

Operadores para incremento e decremento
Operador Descrição

-- Decrementa valor em um.

Operadores condicionais
Operador Descrição
== Verifica se valores da esquerda e direita são idênticos.
< Verifica se valor da esquerda é menor que valor da direita.
> Verifica se valor da esquerda é maior que valor da direita.
<= Verifica se valor da esquerda é menor ou idêntico ao valor da direita.
>= Verifica se valor da esquerda é maior ou idêntico ao valor da direita.
!= Verifica se valor da esquerda não é igual ao valor da direita.

Operadores lógicos
Operador Descrição

|| OU lógico, equivalente ao comando na linguagem C.
! NÃO lógico, equivalente ao comando na linguagem C.

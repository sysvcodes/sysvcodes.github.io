Descrição

Não foram muitos motivos que levaram a criação da linguagem, porém dentre
eles esta a possibilidade de ter certo controle sobre a sintaxe, podendo
desenvolver funcionalidades específicas, deixando a mesma principalmente
mais resumida. Como também desenvolver funcionalidades voltadas para o
cenário da segurança, tornando a linguagem fácil para o desenvolvimento
de ferramentas e realização de tarefas.

Este texto tem como intuito esclarecer algumas questões a respeito da
linguagem, como também servir de referência para quem faz uso da mesma.


Características

  • Compilada
  • Case-sensitive
  • Estruturada


Variáveis

// Definição de várias variáveis na mesma linha.
num1 = null, str1 = null, str2 = null, num2 = null

// Utilização de variáveis de maneira dinâmica.

variavel = null
variavel = alloc 256 // Aloca espaço para 256 bytes.
variavel = "String here!"
variavel = 100


Bibliotecas.

É possível utilizar tanto a biblioteca padrão como bibliotecas externas.
O nome da biblioteca é o mesmo nome do arquivo sem a extensão. Ao
referenciar o nome da mesma o compilador procura o arquivo, se não
existir procura na biblioteca padrão. Para definição utiliza-se o comando
req.

req io // Utilização da biblioteca padrão de entrada e saída.
req minha-lib // Utiliza biblioteca própria.
req dir/lib-A // Utiliza biblioteca própria armazenada em diretório.


Funções

Para declaração de funções usamos o comando def e todo código entre
as chaves { e } é parte da função.

// Exemplo 1
def functionZero{/* Code here... */}

// Exemplo 2
def functionOne
{ /* Code here... */ }

// Exemplo 3
def functionTwo (valor1, valor2) {
     ret valor1 + valor2
}

// Exemplo 4
def
functionThree
(param1, param2, paramN) {
     // Code here...
}

// Exemplo 5
def functionFour
(valora,
valorb)
{
     // Code here...
}

Para retornar dados utiliza-se o comando ret.

// Passagem de parâmetros para função.
def functionHere (valor1, valor2) {
     valor3 = valor1 + valor2
     ret valor3 // Retorna resultado da soma.
}


Ponto de inicialização.

Como na linguagem C, é necessário definir o ponto de inicialização
da aplicação, no caso utilizando a função entry.

def entry {
     // Code here...
}


Controle de fluxo

Comando Descrição
if Se condição for verdadeira executa bloco de código.

if (CONDIÇÃO) {
     // Bloco de código...
}
elif Quando vazio executa bloco padrão (comportamento como do comando else da linguagem C), o bloco apenas é executado se não existir nenhuma ocorrência anterior.

elif {
     //code here!
}

Quando é passado parâmetros, ou seja, algum bloco de condições, a sequência de ifs finaliza com a primeira ocorrência encontrada.

elif (CONDIÇÃO) {
     // code here... (comportamento como
     // do comando else if da linguagem C).
}
for Mesmo funcionamento que o comando na linguagem C, mas com alguns adicionais. O bloco condicional é apenas necessário declara-lo se realmente for utilizar o mesmo. Por exemplo, para um bloco com repetições infinitas podemos fazer da seguinte maneira.

for {
     // code here!
}

Outra característica é a utilização de sua variável interna. Internamente ao utilizar o comando for é declarado uma variável inicializada em 0 e incrementada em 1 a cada execução do bloco de código. Para acessar a mesma basta utilizar o caractere # (jogo da velha).

// Imprime números de 0 a 9 utilizando a
// variável interna do comando for.
for {
     // Code here...
     if (# == 9)
          break;
}

Outros exemplos...
Imprimir números de 0 a 9...

for (a=0; a<=9; a++)
     // Code here...

Utilização de múltiplas variáveis.

for (a=0,b=1,c=2; a<=9; a++,b+=2,c+=3)
     // Code here...
break Encerra a execução do comando for, pulando para o bloco de código seguinte.
continue Pula para início do comando for.
do Quase idêntico ao comando for, com a diferença onde o bloco de código é executado ao menos uma vez antes de verificar o bloco de condições do comando.

do {
     // Code here!
} (CONDIÇÃO);


Acoplamento de código ASM.

Para inserção de código ASM utiliza-se o comando asm. É aceito apenas
sintaxe Intel. O assembler utilizado é o NASM e para linker o programa ld.

asm {
    mov [ebp+28], 10
    mov [ebp+32], 20
}


Opções para linker.

Ao utilizar funções da API do sistema é necessário especificar o nome da biblioteca
que a função pertence.

linker [ lib, kernel32 ]
linker [ lib, user32 ]


Chamadas a funções da API do sistema.

Utiliza-se como na linguagem C, especificando o nome da função e seus
parâmetros.

MessageBoxA NULL, "Minha mensagem!", "Meu titulo!", MB_OK


Operadores, constantes e caracteres especiais.

Caracteres especiais
Caractere Descrição
= Atribuição de valores.
// Comentário de uma única linha ou parte da mesma.
/*, */ Comentário de múltiplas linhas, com início em /* e término em */.
new-line, ; Nova linha e ponto e vírgula, finalização de bloco de comando.
{, } Chaves, define bloco de código, com início em { e término em }.
" (aspas dupla) String, define uma sequência de caracteres, com início na primeira ocorrência encontrada e término na seguinte. Para inserir um aspas dupla dentro da string basta fazer uso da barra invertida antes do caractere (\").
# Acesso a variável interna do comando for e do. Possui valor numérico.

Constantes
Nome Valor
null, NULL 0
true, TRUE 1
false, FALSE 0

Operadores matemáticos
Operador Descrição
+ Soma valores.
- Subtrai valores.
* Multiplica valores.
/ Divide valores.
% Resto de divisão.

Operadores para incremento e decremento
Operador Descrição
++ Incrementa valor em um.
-- Decrementa valor em um.
+= Soma e atribui valor.
-= Subtrai e atribui valor.
*= Multiplica e atribui valor.
/= Divide e atribui valor.
Quando o operador é utilizado antes da variável a mesma é manipulada, digamos,
antes de sua utilização.

Operadores condicionais
Operador Descrição
== Verifica se valores da esquerda e direita são idênticos.
< Verifica se valor da esquerda é menor que valor da direita.
> Verifica se valor da esquerda é maior que valor da direita.
<= Verifica se valor da esquerda é menor ou idêntico ao valor da direita.
>= Verifica se valor da esquerda é maior ou idêntico ao valor da direita.
!= Verifica se valor da esquerda não é igual ao valor da direita.

Operadores lógicos
Operador Descrição
&& E lógico, equivalente ao comando na linguagem C.
|| OU lógico, equivalente ao comando na linguagem C.
! NÃO lógico, equivalente ao comando na linguagem C.
